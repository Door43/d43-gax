====== 19. **//Raagota// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-01.jpg?direct&}}

//Waaqi dabarsii Israa’elii mara keessatti raagota erge. Isaan Waaqarraa ergaa dhage’ane; achiin duuba ergaa Waaqaa san kolba’atti himane.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-02.jpg?direct&}}

//Ahaab Israa’el irratti yennaa mootii te’e, Eliyaas raagaa te’eera. Ahaab nama hamaa akka kolbaan waaqa dharaa ka, <<Ba’aal>> jedhamu akka waaqonfattu tirtisse. Eliyaas, Ahaabiin, <<Ani haga dubbadhutti Israa’el keessatti bokkeenni yookiin fixeensi hin jiraatu>> jedhe. Dubbiin tun akka malee Ahaab aarsite.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-03.jpg?direct&}}

//Waaqi namicha isa ijjeesiisaaf barbaadu Ahaab duraa isa dhossiisaaf, akka inni gara laga goomolee keessaa dhaqu Eliyaasitti hime. Ganama ganamaa fi galgala galgala mara harraaqessi sagalee isaaf fidaa turane. Ahaabii fi loltooti isaa Eliyaas barbaadane; te’uu malee isa dhagguu hin dandeenne. Olaan guddoo hamaa waan te’eef dhuma irratti lagi sun goge. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-04.jpg?direct&}}

//Tanaaf, Eliyaa gara biyya ollaa isaa takka dhaqe. Biyya san keessatti sababa beelaatiif, haadha hiyyeessaa takkaa fi ilma isi’ii sagaleen duraa dhumatiisa geetteertiiyyu. Te’uu malee isaan Eliyaas keessummeessane; Waaqille bullaan ibidoo isaanii keessaa akka hin dhumanne, sayitiille okkotee isaanii keessaa akka dhidhanne godhee isaan areere. Bara beelaa san mara sagaleen isaan duraa hin dhumanne. Eliyaas woggaa hedduu achi le’e.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-05.jpg?direct&}}

//Woggaa sadii fi wodha’aan duuba akka bokkeya roobisiisu gara mootummaa Israa’elii deebi’ee Ahaabitti akka himu Waaqi Eliyaasitti hime. Ahaab yennaa Eliyaas dhagge, <<Ati nama rakkinna uumu, dhuttee motii!>> jedheen. Eliyaas deebisee, <<Nami rakkinna uumu si’i! Waaqa dhuga’aa, Mootii Waan Maraa lakkitteerta; ba’aal waaqonfateertaa, kolbaa mootummaa Israa’elii gaara qeremelosii irratti wolitti naa qabi>> jedheen. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-06.jpg?direct&}}

//Raagota ba’aalii 450 woliin kolbaan mootummaa Israa’elii marti gara gaara qeremeloosii dhutte. Eliyaas kolba’aan, <<Isin yennaa meeqa yaada keessan jijjiirattan? Mootiin Waan Maraa Waaqa yoo te’e isa tajaajila’a! Ba’aal Mootii Waan Maraa yoo te’e isa tajaajila’a>> jedheen. Achiin duuba Eliyaas raagota ba’aaliitiin, <<Jibicha tokkocha qala’aatii ciincaa dhikeessa’a; te’uu malee ibidda hin gabbisina’a. Anille akkasuma hin godha. Waaqi ibiddaan deebii kennu inni Waaqa dhuga’aa ti>> jedhe. Tanaaf, hayyooti ba’aalii ciincaa qopheessane. Te’uu malee ibidda hin gabbinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-07.jpg?direct&}}

//Achiin duub raagoti ba’aalii , <<Ee ba’aal nuu dhage’i!>> jedhanee gara ba’aalii daadimatane. Guyyaa guutuu daadimatane; iyyane; nafa ifiille billowaan mummurane; te’uu malee deebii dhabane.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-08.jpg?direct&}}

//Dhuma irratti Eliyaas ciincaa Waaqaaf dhikeesse. Achiin duuba foon, muki, akkasuma latti adaala foon sanii jirtu marti akka malee haga shantutti, bisshaan ibiddoo kudhanii lama guutuu foon ciinca’aa san irratti akka dhangalaasanu kolba’atti hime. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-09.jpg?direct&}}

//Achiin duuba Eliyaas, <<Ee Yaahiwwee, Waaqa Abrahaamii, Waaqa Yisihaaqii fi Yaaqoobii; ati Waaqa Israa’elii akka teetee fi anille tajaajilaa keeti akka te’e nutti mudhis. Ati Waaqa dhuga’aa akka teete kolbaan tun akka beettuuf, deebii naa kenni>> jedhee daadimate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-10.jpg?direct&}}

//Yoosuma ibiddi ol-gubba’aa bu’ee foon, muka, dhakaa, huura, akkasuma bisaan adaala addee ciinca’aa jiruuyyuu gube. Kolbaan yennaa waan kana dhaggite, lafatti gad jiddee, <<Yaahiwween, Waaqa! Yaahiwween. Waaqa!>> jette.   //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-11.jpg?direct&}}

//Achiin duuba Eliyaas, <<Raagota ba’aalii keessa tokkochille hin bayin>> jedhe. Tanaaf kolbaan sun raagota ba’aalii san qaddee geessitee ijjeette.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-12.jpg?direct&}}

//Achiin duuba Eliyaas Ahaabii mootichaan, <<Bokkeenni dhufeeraa, ariifadhuu gara qacha’aa deebi’i>> jedheen; gubbaan yoosuma gurraachotte. Baradaan roobuu jalqabe. Yaahiwween oolaa dhaabbachiisee, Waaqa dhuga’aa akka te’e mirkaneesse.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-13.jpg?direct&}}

//Bara Eliyaasiitiin duuba Waaqi nama Elisaa’i jedhamu akka raagaa te’u file; karaa Elisaa’iitiin Waaqi bilbaasa hedduu huje. Bilbaasota san keessaa tokkochi dhukkub goga’aa farrisiisaan ajajaa loltoota diinaa ka te’e Ni’imaan irratti mudhate. Marroo Elisaa’ii waan dhage’eeruuf, akka inni isa fayyisuuf dhaqee Elisaa’i gaafate. Elisaa’i akka Ni’imaan bisaan laga Yordaanosii yennaa torba lixee  fulla’u itti hime.  //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-14.jpg?direct&}}

//Jalqaba irratti Ni’imaan mufatee, gowwummaa waan itti fakkaatteef, waan Elisaa’i itti hime hin goone. Duubarra ammoo yaada ifii jijjiiratee, bisaan laga Yordaanosii yennaa torba lixee fulla’e. Dhuma irratti yennaa inni bisaan keessaa baye gogaan isaa ijjumaa fayye. Waaqi isa fayyise.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-15.jpg?direct&}}

//Waaqi raagota dhibii hedduu erge. Nami marti waaqota dharaa waaqonfatiisa akka lakkisuu fi worra dhibi’iif haqa akka huju, araaralle godhuu akka jalqabanu kolba’atti hime. Waan hantuu hujiisa yoo hin lakkisin, Waaqaaf ajajamiisalle yoo hin jalqabin, Waaqi yakkaamessumma’aan akka isaanitti muru, akka isaan adabulle worri raagota kolbaa ifi eeggachiisane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-16.jpg?direct&}}

//Yennaa hedduu kolbaan Waaqaaf hin ajajanne. Yennaa hedduu haalli isaan raagota itti qabatanulle haala bada’a.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-17.jpg?direct&}}

//Yennaa tokko tokko isaan raagota hin ijjeesaniiyyu. Yennaa takka, raagicha Ermiyaas akka inni boolla bisaanii godduu keessatti gatamee achitti du’u lakkisane. Inni dhoqqee boolla bisaanii golba jirtu keessa seene. Te’uu malee yennaa san mootichi isaaf nayee, adoo inni hin du’in, boolla san keessaa harkisanee akka isa baasanu tajaajiltoota ifii ajaje.////.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-19-18.jpg?direct&}}

//Raagoti sun kolbaan yoo isaan jibbitelle marroo Waaqaa dubbatiisa hin lakkinne. Isaan yoo araara hin galin Waaqi akka isaan balleessu kolbaa ifi eeggachiisane. Masiihiin Waaqaa akka dhufulle addii inni kenne kolbaan akka qaabattu godhane.//

**//Dabarsii Kitaaba Woyyichaa:- 1Moototaa 16-18; 2Moototaa 5; Ermiyaas 38.// **
