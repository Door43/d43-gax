====== 12. **//Israa’elooti Biyya Gibxi’ii Keessaa Bayane// ** ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-01.jpg?direct&}}

//Israa'elooti biyya Gibxi’ii keessaa waan bayaneef guddoo gammadane. Isaan haganaa achi garboole'ee moti. Isaan gara lafa waadaan isaaniif galamee ideemiisatti jiraniiyyu. Gibxooti worqi'ii fi meetii, wonni gatii guddo'oo dhibiille adoo hin hafin, waan isaan gaafatane mara Israa'elootaaf kennanee, yennaa isaan biyya Gibxi’ii keessaa bayanu, gosa dhibii keessaa worri Waaqatti dhugeeffate tokko tokko Israa'eloota woliin deemane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-02.jpg?direct&}}

//Waaqi gaafa isaaniin dura deemu, guyyaa utubaa duumensaa dheera'aan, halkan utubaa ibiddaa dheera'aan isaan sooresse. Waaqi yennaa mara isaan woliin waan jiruuf, yennaa isaan deemanu isaan sooresse. Wonni isaan godhiisaaf malane isa gula deemiisa qofa. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-03.jpg?direct&}}

//Guyyaa diqqa'aan duuba Fara'oonii fi kolbaan isaa yaada ifii jijjiiratanee, ammalle Israa'elooti garboota isaanii akka te'anu barbaadane. Waaqi Mootiin Waan Maraa dhugicha tokkocha akka te'e kolbaan akka dhaggituuf, Mootiin Waan Maraa, Fara'oonii fi waaqota isaa caalaa hunnaamessa akka te'elle akka hubatanuuf, inni gadhaa Faara'onii jabeesse. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-04.jpg?direct&}}

//Maarre Fara'oonii fi loltooti isaa ammalle garboota isaan godhatiisaaf Israa'eloota hordofane. Israa'elooti loltooti Gibxootaa isaan hordofanuun yennaa dhaggite, loltoota Fara'ooniitii fi abbaayaa diintuu oddu’utti akka qabamane beekanee, guddoo waan sodaataneef, <<Biyya Gibxi’ii keessaa maaf baane? Amma laa duune>> jedhanee iyyane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-05.jpg?direct&}}

//Muuseen Israa'elootaan, <<Hin sodaatina'a! Waaqi adha isiniif lolee isin hin hambura>> jedhe. Achiin duuba Waaqi Muuse'een, <<Gara abbaayaa diintu'uu akka deemanu kolba'atti him>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-06.jpg?direct&}}

//Achiin duuba Waaqi Gibxooti Israa'eloota dhaggiisaaf akka hin dandeenne Israa'elootaa fi Gibxoota wodhakaa utubaa duumensaa keye.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-07.jpg?direct&}}

//Waaqi akka Muuseen harka ifii abbaayaa irra diriirsee bisaan gargar kutu itti hime. Achiin duuba Waaqi abbaayaa keessa karaan dabaranu akka jiraatuuf, qilleensi akka bisaan bita'aa fi middatti dhaabbachiisu godhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-08.jpg?direct&}}

//Israa'elooti bisaan bita'aa fi midda isaaniititti akka dhaaba manaa isaaniif te'ee, isaan lafa godduu irra abbaayyaa keessa deemane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-09.jpg?direct&}}

//Achiin duuba Waaqi Gibxooti akka Israa'elooti isaan jalaa bayane akka dhagganuuf, duumensa karaa irraa ol kaase. Gibxooti isaan duubaan hordofiisaaf murteeffatane.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-10.jpg?direct&}}

//Maarre isaan karaa abbaaya'aatiin Israa'eloota hordofane. Waaqi ammoo Gibxooti akka rakkattuu fi garreettaan isaanii ka fardaan harkfamu akka hidhamu godhe. Isaan, <<Baqadha’a! Waaqi Israa'elootaaf loliisatti jira>> jedhanee iyyane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-11.jpg?direct&}}

//Israaelooti nageyaan abbaayaa gamatti bu'aneen duuba, Waaqi ammalle Muuseen harka ifii akka diriirsu itti hime. Innille yennaa ajajame, bisaan loltoota Gibxootaa irratti garagalee akka qaraa kaan te'e. Loltooti Gibxootaa marti liqinfamane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-12.jpg?direct&}}

//Israa'elooti akka Gibxooti du'ane yennaa dhaggane Waaqatti dhugeeffatane. Muuseelle raagaa Waaqaa akka te'e dhugeeffatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-13.jpg?direct&}}

//Waaqi du'aa fi garbummaa jalaa waan baaseef, Israa'elooti ammalle guddoo gammadane. Amma Waaqa tajaajiliisaaf bilisa bayane. Israa'elooti waan Waaqi loltoota Gibxootaa jalaa waan isaan baaseef, bilisummaa haareya ifii jilfatiisaaf, Waaqalle galateeffatiisaaf, faaruu bacaa faarsane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-12-14.jpg?direct&}}

//Gibxoota irratti injifannoo isaaniif akka kennee fi garbummaa jalaa akka isaan baase akka qaabatanuuf, wogguma wogga'atti Jila Irra Dabaramaa akka jilfatanu, Waaqi Israa'eloota ajaje. Isaan buruusoo fafa hin qanne qalatanee daabboo uukoo hin qanne woliin nyaatanee gammadane. //

**//Dabarsii Kitaaba Woyyichaa:- Seera Bayiisaa 12:33–15:21 // **
