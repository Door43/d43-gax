====== 10. **//Balaa Kudhan// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-01.jpg?direct&}}

//Muuse’ee fi Aaron gara Faara’onii dhaqanee, <<Waaqi Israa’elii, <Kolbaa tiyya lakkis> siin jedha>> jedhaneen. Fara’oon garuu isaan hin dhageenne. Israa’elooti akka bilisa bayanu lakkisiisa irra hujii hunnaa akka hujanu godhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-02.jpg?direct&}}

//Fara'oon ammalle kolbaa lakkisuu didate. Tanaaf, Waaqi biyya Gibxi’ii irratti balaa kudhan erge. Waaqi karaa balaa kanaatiin waaqota Faara'oniitii fi Gibxootaa mara caalaa hunnaamessa akka te'e isaanitti mudhise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-03.jpg?direct&}}

//Waaqi bisaan Nayilii gara dhiigaa jijjiire. Fara'oon ammoo ammalle Israa'eloota gad hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-04.jpg?direct&}}

//Waaqi Gibxoota mara irratti raacha erge. Fara'oon akka Muuseen raacha isa irraa fageesu kadhate. Te'ulle, te’uu dhabaatulle raachi marti dhumate. Fara'oon gadhaa jabaatee Israa'elooti biyya Gibxi'ii keessaa akka bayanu hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-05.jpg?direct&}}

//Tanaaf, Waaqi faacha erge. Achiin duuba tiittee erge. Fara'oon Muuse'ee fi Aaron waamee, yoo isaan balaa kana dhaabbachiisane, Israa'elooti akka deemiisa dande'anu isaanitti hime. Muuseen yennaa daadimate Waaqi tiittee mara kolbaa irraa balleesse. Fara'oon ammoo gadhaa jabaatee kolbaa gad hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-06.jpg?direct&}}

//Itti aansee Waaqi horiin Gibxootaa marti dhibamanee akka du'anu godhe. Garuu Fara'oon gadhaa jabaatee Israa'eloota gad hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-07.jpg?direct&}}

//Achiin duuba Waaqi Muuseen Fara'oon duratti daadhaa gara ol-gubba'aa akka bittinneesu itti hime. Yennaa inni daadhaa bittinneesse Gibxoota irratti madaan dukkubu isaan irratti te'e. Israa'eloota irratti garuu madaan hin teene. Waaqi gadhaa Fara'oonii jabeesse. Fara'oonille Israa'eloota gad hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-08.jpg?direct&}}

//Achiin duuba Waaqi dhakaa bokkeyaa ka biqiltuu biyya Gibxi’ii keessaa mara balleessee nama manaan ala jiru mara ijjeese erge. Fara'oon Muuse'ee fi Aaaron waamee, <<Ani yakkeera! Deemuu hin dandeettan>> jedhee isaanitti hime. Maarre Muuseen daadimatee dhakaan bokkeyaa ol-gubbaa irraa roobuu lakkise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-09.jpg?direct&}}

//Garuu Fara'oon ammalle cubbuu huje; gadhaalle jabaate. Israaeloota gad hin lakkinne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-10.jpg?direct&}}

//Tanaaf, Waaqi biyya Gibxi’ii irratti awwaanisa fide. Awwaannis kun midhaan dhakaan bokkeyaa hin balleeesin mara nyaatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-11.jpg?direct&}}

//Achiin duuba Waaqi dukkana guyyaa sadii erge. Gibxooti  mana ifiitii gad bayiisa haga dhabanutti guddoo dukkanootte. Addee Israa'elooti le’anutti ammoo ifi jiraayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-10-12.jpg?direct&}}

//Balaa sagal kanaan duuballe Fara'oon ammalle Israa'eloota gad hin lakkinne. Fara'oon marroo dhage'iisa didateef, Waaqi balaa dhumaa tokko ergiisaaf murteeffate. Kun yaada Fara'oonii hin jijjiira. //

**//Dabarsii Kitaaba Woyyichaa:- Seera Bayiisaa 5-10 // **
