====== 3. **//Bisaan Badi’ii// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-01.jpg?direct&}}

//Bara // heddu'uun// duuba kolbaan hedduun biyya lafaa kessa le'ane. Isaan worra hamaaleyyi'ii fi hunnaameyyii te'ane. Hammeenni bacataa waan dhufeef, Waaqi bisaan badi'ii ka lola'ee balleessuun biyya lafaa balleessiisaaf murteeffate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-02.jpg?direct&}}

//Garuu Nohi Waaqa duratti // fudhatama dhaggate. Nohi namoota bara san jiru mara caalaa, nama balcha'aa fi nama yakkaa irraa fagaateeru .Waaqi bisaan badi'ii ka fidiisaaf karoora baafate san Nohitti hime. Nohi hoboloo guddittii akka huju Waaqi isatti hime.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-03.jpg?direct&}}

Waaqi akka Nohi hoboloo guddittii huju itti hime. <<Dheerinna isi'ii meetirii dhibba tokkoo fi afurtama, badhinna isi'ii meetirii diddamii sad, hojjaan isi'ii meetira kudha sadii fi wodhakkaa teetuu ti>> jedhe. Hoboloon guddittiin Nohi huje ta muka irraa hujante teetee, ta mana mana irratti ijaarame sad, kolloo bacaa, heellicho’oo fi fooddaa qaddu. Hoboloon guddittiin sun Nohii fi maatii isaa, bineensota lafa irraa ka gosa adda adda mara bisaan badi’ii irraa akka hanburtu itti hime.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-04.jpg?direct&}}

Nohille Waaqaaf ajajamee akkuma Waaqi isaanitti hime, Nohii fi ilmaan isaa sad hoboloo guddittii san hujane. Hoboloon guddittiin sun akka malee guddoo waan teeteef, inni adoo isii hujuu woggaa hedduu fixe. Nohi marroo bisaan badi'ii ka dhufaaruu, kolbaa ifi eeggachiisaa, gara Waaqaa akka deebi'aanu isaanitti hime; te'uu malee isaan isa hin dhugeeffanne.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-05.jpg?direct&}}

Waaqi Nohii fi maatiin isaa, ifii fi bineensotaafille saagalee isaan geettu akka tuulatanu ajaje. Wonni marti yennaa qophootte, hoboloo guddi'itti akka galanu, Nohitti, niitii isaatitti, ilmaan isaa sadii fi niitota isaaniititti, yennaan akka geette hime. Namooti saddeeti hoboloo guddittii sanitti timane.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-06.jpg?direct&}}

Waaqi akka isaan bisaan badi'ii jalaa bayanu dhala'aa fi korma waan maraa, sinbirrootalle dabalatee, gara Nohii isaan erge. Bineensota gosa adda addaa mara keessaa ciinca'aaf akka te'anu kormaa fi dhalaa torba torba gara hoboloo guddittii Nohii san erge. Marti isaaniituu hobooloo guddittii keessanitti akkuma timaneen Waaqi ifumaaf balbala hoboloo guddittii sanii cufe.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-07.jpg?direct&}}

Achiin duuba bokkeenni roobuu jalqabbe; roobee, roobee, roobee adoo gargar hin kutin guyyaa afurtamaa fi halkan afurtama roobe. Lafa keessalle'ee maddi burqe. Wonni biyya lafaa irraa marti bisaaniin liqinfamane; gaarrri dhedheeraalle adoo hin hafin bisaaniin liqinfame.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-08.jpg?direct&}}

Namaa fi bineensota hoboloo guddittii san keessa jiranu malee, wonni lafa irraa marti dhumatane. Hoboloon guddittiin sun bisaan irratti ol baatee, waan isii keessaa mara du’a irraa hanburte.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-09.jpg?direct&}}

Bokkeenni caameen duuba hoboloon guddittiin sun haga ji'a shanii bissaan gubbaa teette. Yeenna san bisaan dhidhatiisa jalqabe. Gaafa tokko hoboloon guddittiin sun qaccee gaaraa irra teette; te'uu malee latti duudiin ammalle bisaaniin liqinfanteertiiyyu. Ji'a sadiin duuba qacceen gaarotaa mudhatiisa jalqabane.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-10.jpg?direct&}}

Guyyaa afurtamaan duuba, Nohi bisaan lafa irraa akka gogeeru ilaaliisaaf harraaqessa erge. Harraaqessi sun asii fi achi koolaan ka'ee, lafa godduu yennaa barbaade dhaggiisaaf hin dandeenne.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-11.jpg?direct&}}

Achiin duuba Nohi saphaliisa tokko erge. Te'uu malee saphaliis sun addee itti tee'u waan dhabeef gara hoboloo Nohii deebi'ee dhufe. Torbaan tokkoon duuba Nohi ammalle saphaliisa san erge. Saphaliis sun kichuu ejersaa afaanitti qabattee deebi'e; bisaan dhidhataa waan deemeef, biqiltooti biqiluu jalqabane.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-12.jpg?direct&}}

Nohi torbaan// tokko turee yennaa sadeesso'oo saphaliisa erge. Yennaa tana addee itti tee'u waan dhaggatteef deebi'ee hin dhunne. Bisaan gogiisatti jiraayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-13.jpg?direct&}}

//Ji’a lamaan duuba Waaqi, Nohiin, <<Ati, maatiin keetii fi bineensotille hoboloo // guddittii san keessaa gad baya’a>> jedhe. Maatiin isaa hoboloo guddittii san keessaa gad bayane. Ijoollee baca’aa fi akaakota bacaa horanee lafa guutane.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-14.jpg?direct&}}

Nohi hoboloo guddittii keessaa bayeen duuba addee ciinca'aa ijaaree bineensota gosa addaa addaa// ka Waaqaaf ciinceffamiisa dande'anu Waaqaaf ciincaa dhikeesse. Waaqille ciincaa sanitti gammadee Nohii fi maatii isaa ebbise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-15.jpg?direct&}}

//Waaqi, <<Waan hantuu // namooti hujannuuf jedhee deebi'ee biyya lafaa ijjumaa akka hin balleesine waadaa si'iif gala; namooti ijoollumaa ifitiin jalqabanee cubbaaleyyii te'anule, ani biyya lafaa deebi'ee bisaan badi'iitiin hin balleessu>> jedhe.

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-03-16.jpg?direct&}}

Waaqi waadaa galeef beessisa akka teetu hiituu Waaqaa ta jalqabaa dade. Yennaa mara hiituun Waaqaa// gubbaa irratti mudhatti. Waaqi karaa hiituu ifii tanaan waadaa kolba'aaf gale san qaabata. //
