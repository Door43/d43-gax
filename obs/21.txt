====== 21. **//Waaqi Masiihii Addii Kenne// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-01.jpg?direct&}}

//Jalqaba irrattuu, Waaqi Masiihii ergiisaaf koroorfate. Kennamiis Addii Masiihii ka jalqabaa gara Addaamii fi Heewwanii dhufe. Ka mataa bofaa butuchu sayniin Heewwanii akka dhalatu Waaqi addii kenne. Bofi Heewwan sobe Seexana. Ka addii te’e Masiihiin Seexana ijjumaa hin injifata jechu’u.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-02.jpg?direct&}}

//Karaa isaatiin gosi kolbaa biyya lafaa marti eebba akka fudhattu Waaqi Abrahaamiif addii kenne. Eebbi kun gara duraa yennaa Masiihiin dhufe hin muummowa. Inni kolbaa biyya lafaarra jirtu gosa maraayyuu akka isaan fayyanu hin godha. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-03.jpg?direct&}}

//Gara duraa raagichi akka Muuse’ee jiru akka dhufu’uu jiru Waaqi Muuse’eef addii kenne. Kun marroo Masiihii turee dhufuu addii kennante ta dhibi’i.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-04.jpg?direct&}}

//Sanyii isaa keessaa tokkochi kolbaa Waaqaa irratti mootii te’ee haga bara baraa akka bulchu’uu jiru mooticha Daawitiif Waaqi addii kenne. Kunille Masiihiin sanyii Daawitii keessaa tokkocha te’a jechu’u.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-05.jpg?direct&}}

//Waaqi Gondooroo Haareya karaa Ermiyaasii raagichaatiin akka godhu addii kenne; Gondooroon tun ammoo Gondooroo akka Waaqi gaara Siina’aa irratti kolbaa Israa’elii woliin godhee moti. Gondooroo Haareyaan Waaqi seera ifii qalbii kolba’aatitti barreessa. Kolbaan ifumaan Waaqa beetti; kolbaa isaa hin teeti; Waaqi cubbuu isaanii hin araarama. Masiihiin Gondooroo Haareya hin jalqaba.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-06.jpg?direct&}}

//Ammalle raagoti Waaqaa Masiihiin raagicha, hayyicha,  mootichalle hin te’a jedhane. Raagichi nama jecha Waaqaa dhage’u; achiin duuba jecha Waaqaa kolba’aaf ka lallabu. Masiihiin Waaqi ergiisaaf addii kenne raagicha komii hin qanne hin te’a. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-07.jpg?direct&}}

//Hayyooti Israa’elootaa marroo cubbu’uutiif adabbii kolbaa irra dhaqqaddu bakka buusanee Waaqaaf ciincaa dhikeessane. Marroo kolba’aatiifille gara Waaqaa daadimatane. Masiihiin guutumatti mataa ifii ciincaa godhee ka Waaqaaf dhikeessu  hayyicha guddaa komii hin qanne hin te’a.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-08.jpg?direct&}}

//Mootiin ka mootummaa bulchuu fi nama kolbaa irratti muru. Masiihiin sanyii isaatiin ka barcumaa mootumma’aa irra tee’u mootii komii hin qanne hin te’a. Guutummaa biyya lafaa irratti haga bara baraa mootii hin te’a. Inni haqaan waan muruuf, murtee sirri’ii kenna. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-09.jpg?direct&}}

//Raagoti Waaqaa marroo maasiihii waan hedduu raaganeeran. Raagichi Milkiyaas Masiihiin adoo hin dhufiniin dura raagichi goddaan dursee akka dhufu dursee raage. Masiihiin intala durraa takka irraa akka dhalatu raagichi Isaayaas raage. Raagichi Mikiyaas Masiihiin qachaa Beetelemii keessatti hin dhalata jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-10.jpg?direct&}}

//Raagichi Isaayaas Masiihiin Galiilaa keessa le’a; namoota guddoo gaddane hin jajjabeessa; akkasuma worra booji’ameeruuf bilisummaa, worra hidhameeruufille hiikamiisa hin lallaba. Ammalle raagichi Isaayaas Masiihiin worra dhibamee fi kolbaa dhage’iisa, dhaggiisa, dubbatiisa yookiin deemiisa hin dandeenne akka fayyisu raage.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-11.jpg?direct&}}

//Ammalle raagichi Isaayaas sababa malee Masiihiin akka jibbamuu fi mormiin akka irra dhaqqaddu raage. Raagoti dhibiin worri Masiihii ijjeese woyaa isaa irratti hixaa akka buufanuu fi jaalli isaa akka isa mormatu dursanee raagane. Raagichi Zakaariyaas jaala Masiihii mormatuuf meetii soddoma akka kaffalanuuf raage. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-12.jpg?direct&}}

//Ammalle raagichi marroo du’iisa Masiihii dubbate. Isaayaas namooti Masiihii irratti akka tufanu, akka shasheessanuu fi akka isa dikanu raage. Inni dokonkora tokkolle ka hin hujin te’ulle isa hin woraanan; dhiboo guddo’oo fi farra goddo’oon hin du’a.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-13.jpg?direct&}}

//Ammalle raagoti, << Misiihiin ka  cubbuun irra hin jirre ka dhidhinna hin qanne hin te’a>>  jedhane. Adabbii cubbuu namoota dhibi’ii fudhatiisaaf inni du’e. Adabamiis isaa Waaqaa fi namoota wodhakka’atti nageya fida. Sababa kanaaf, Masiihii butuchuun fedhi Waaqaa te’e. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-14.jpg?direct&}}

//Masiihiin akka du’uu fi Waaqi ammoo du’a keessaa akka isaa kaasu raagoti dursanee raagane. Waaqi karaa du’aa fi ka’a Masiihiitiin cubbaaleyyii fayyisiisaaf karoora ifii muummessee, Gondooroo ifii haareya hin jalqaba. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-21-15.jpg?direct&}}

//Waaqi worra raagotaatitti marroo Masiihii waan hedduu itti mudhise; te’uu malee, Masiihiin bara raagoti sun jiranu keessa hin dhunne. Raagota san keessaa ka dhuma irraa raagee woggaa dhibba afuriin duuba, yennaa isi’iititti, Waaqi Masiihii gara biyya lafaa erge.    //
