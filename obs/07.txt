====== 7. **//Waaqi Yaaqoob Eebbise.// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-01.jpg?direct&}}

//Ijoolleen sun yennaa guddate, Yaaqoob haadha woliin mana ooliisa jaalataayyu. Esaawu ammoo adamoo adansuu jaalataayyu. Ribqaan Yaaqob jaalattiiyyu.Yisihaaq ammoo Esaawu jaalataayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-02.jpg?direct&}}

//Gaafa tokko Esaawu adamoo irraa yennaa deebi'u beelote. Esaawu Yaaqoobiin, <<Maganne'e, sagalee qopheessite irraa diqqoo isi'ii naa kenni>> jedhe. Yaaqoob deebisee, <<Atille angafummaa teeti naa kenni>> jedheen. Maarre Esaawu angafummaa ifii Yaaqoobiif kenne. Achiin duuba Yaaqoob sagalee qopheesse irraa isaaf kenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-03.jpg?direct&}}

//Yisihaaq eebba ifii Esaawuuf kenniisa barbaade. Garuu adoo inni tana hin tolchiniin dura Ribqa'aa fi Yaaqoob Yisihaaq sobane. Yaaqoob Esaawu fakkaatee gara Yisihaaqii dhikaate. Yisihaaq guddoo waan dulloomeeruuf ilaan dhaggiisa hin dande'uuyyu. Maarre Yaaqoob woyaa Esaawuu uffatee, gogaa re'e'ee mormaa fi harkatti hidhatee gara Yisihaaqii dhikaatee isa sobe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-04.jpg?direct&}}

//Yaaqoob gara Yisihaaqii dhufee, <<Kunoo, ani Esaawu dhufeeraa na eebbisuu hin dandeetta>> jedheen. Yisihaaq rifeensa re'e'ee qaqqabee, foolee woya'aalle ulatee Esaawu isattii waan fakkaateef isa eebbise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-05.jpg?direct&}}

//Yaaqoob isa sobee angafumma'aa fi eebba isaa isa irraa waan fudhateef Esaawu isa jibbee, abbaan isaanii du'een duuba isa ijjeesiisaaf yaadate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-06.jpg?direct&}}

//Garuu Ribqaan karoora Esaawuu dhageettee, Yisihaaq woliin mari'attee Yaaqoob fira isi'ii woliin akka le'uuf gara lafa fago'oo isa ergite. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-07.jpg?direct&}}

//Yaaqoob bara hedduu worra Ribqa'aa woliin le'e. Achiin dubaa inni fuudhee ilmaan dhiiraa kudhanii lamaa fi intala takkaa uumate. Waaqi guddoo isa duroonse. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-08.jpg?direct&}}

//Yaaqoob qara Kana'aan keessa eega le'e woggaa diddamaan duuba, maatii ifii, tajaajiltoota ifiitii fi horii ifii woliin gara Kanaa'aanii deebi'isaaf murteefate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-09.jpg?direct&}}

//Yaaqoob, <<Esaawu ammalle na ijjeesiisaaf yaadata fa'a>> jedhee gadhaa ifii keessatti waan herregeef sodaate. Tanaaf horii bacaa Esaawu'uukennansaaf erge. Tajaajiltooti Yaaqoobii worri horii isaaf fidane Esaawu'uun, <<Tajaajilaan keeti Yaaqoob horii kana si'iif ergeera; innille yennaa dhikoo tanatti hin dhufa>> jedhane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-07-10.jpg?direct&}}

//Esaawu garuu qaraayyuu Yaaqoobiif waan araarameeruuf, lammoo isa dhaggiisaaf gammade. Yaaqoob lafa kana'aanii keessa nageyaan le'e. Achiin duuba Yisihaaq du'ee Yaaqoobii fi Esaawu isa awwaalane. Waadaan qara Waaqi Abrahaamii fi gale amma Yisihaaq irraa gara Yaaqoobii dabare. //

**//Dabarsii Kitaaba Woyyichaa: Seera Dadaa 25:27-33:20 // **
