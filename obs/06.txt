====== 6. **//Waaqi Niitii Yisihaaqiif qopheesse// **//. //  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-01.jpg?direct&}}

//Abrahaam yennaa guddoo dulloome, Yisihaaq fuudha geye. Maarre Abrahaam tajaajiltoota ifii keessaa tokkocha, ilma ifii Yisihaqiif niitii barbaadee akka fiduuf gara biyya gosa ifii erge. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-02.jpg?direct&}}

//Tajaajilaan sun yennaa hedduu ideemeen duuba biyya gosi Abrahaamii le'anu geye. Waaqi tajaajilaa Abrahaamii ka ergame san gara Ribqa'aa akeeke. Ribqaan intala ilma obboleessa Abrahaamii ti. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-03.jpg?direct&}}

//Ribqaan worra ifii lakkittee tajaajilaa san woliin gara mana Yisihaaqii deemiisaaf eeyyante. Akkuma achi geetteen Yisihaaq isii fuudhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-04.jpg?direct&}}

//Yennaa heddu’uun duuba Abrahaam du’ee, Waadaan Waaqi Abrahaamiif gale marti gara Yisihaaqii dabarte. Sanyii laakkowanee hin dandenne akka qabaatu Waaqi Abrahaamiif waadaa galeeraayyu. Te’uu malee niitiin Yisihaaqii Ribqaan ijoollee deyiisa hin dandeenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-05.jpg?direct&}}

//Yisihaaq Ribqa'aaf daadimatee, isiin lakkuu akka ulfoottu fedha Waaqaa teete. ijoolleen lamaan gadaamessa Ribqa'aa keessa adoo jiranuu, achuma keessatti woldhaantoo wol qabaniiyyu. Ribqaan wonni gadaamessa isi'ii keessatti te'iisatti jiru kun maan akaka te'e Waaqa gaafatte . //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-06.jpg?direct&}}

//Waaqi Ribqa'aan, <<Ilmaan lamaan gadaamessa keeti keessa jiranu keessaa gosi adda addaa lama hin dhalatti; isaan lamaanuu woldhaantoo wol hin qabatan; angafichi maandhicha hin tajaajila>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-06-07.jpg?direct&}}

//Ilmaan Ribqa'aa yennaa dhalatane, ilmi angafichi diimaa dabbasicha te'ee dhalate; isaan, <<Esaawu>> jedhanee maqaa baasane. Ilmi maandhichille jimuu Esaawuu qabatee dhalate; isallee, <<Yaaqoob>> jedhanee maqaa baasane. //

**//Dabarsii Kitaaba Woyyichaa:- Seera Dadaa 24:1-25; 26. // **
