====== 15. **//Lafa Waada’aa// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-01.jpg?direct&}}

//Dhuma irratti yennaan Israa'elooti gara lafa waada’aa gara kana'aanii galanu geette. Iyaasuun gara  qachaa kana'aanotaa ka, <<Yerikoo>> jedhamuu ka dalleya dhaka’aa jabaa qabuu, dootuu lamaan erge. Sagaagaltittiin, <<Rahaab>> jedhantu ta worra dooyaa dhaqe san dhossiteen duuba akka isaan milqanu qarqaate takka qachaa san keessa leetiiyyu. Isiin marroo Waaqatti dhugeeffatteef tana goote. Worri dooyu sun yennaa Israa'elooti Yerikoo balleessanu Rahaabii fi maatii isi'ii akka baasanu waadaa isi'iif galane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-02.jpg?direct&}}

//Israa'elooti lafa waada’aa sanitti galiisaaf laga Yordaanosii gamatti buu'iisa qabaniiyyu. Waaqi Iyasu'uun, <<Qara hayyooti akka buutu godhi>> jedheen. Hayyooti lagaYordaanosii bu'iisa yennaa jalqabane, Israa'elooti lafa godduu keessa gamatti akka bu'anu, bisaan gara gubba'aatiin lola'iisa lakkise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-03.jpg?direct&}}

//Kolbaan laga Yordaanosii gamatti buuteen duuba Waaqi qachaa Yeriko'oo hunnaamettii san attam akka lolu Iyaasu'utti hime. Kolbaan Waaqaaf ajajante. Loltootii fi hayyooti akkuma Waaqi isaanitti hime guyyaa tokkotti yennaa tokko, haga guyyaa jaa qachaa Yeriko'oo irra naannowane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-04.jpg?direct&}}

//Achiin duuba guyyaa torbeesso'oo, Israa'elooti yennaa torba qachaa irra naannowane. Dhuma irratti qachaa irra naannowiisatti adoo jiranuu loltooti iyyite; hayyootille tultullaa afuutte. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-05.jpg?direct&}}

//Achiin duuba dalleenni dhaka’aa ka Yeriko'otti naannowee jiru jige. Akkuma Waaqi ajaje, Israa'elooti waan qachaa san keessa jiru mara balleessane. Gosa Israa'elootaa ta teete Rahaabii fi maatii isi'ii qofa baasane. Kolbaan dhibiin ta Kana'aan keessa leetu Isaraa'elooti akka Yerikoo balleessane yennaa dhage'ane, Israa'elooti akka isaanille lolanu herreganee sodaatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-06.jpg?direct&}}

//Waaqi akka Israa'elooti gosa kolbaa Kana'aan keessa leetu mara woliin wolii galtee nageyaa hin godhane isaan ajajeeraayyu. Garuu gosa kolbaa Kana'aanotaa keessaa tokko ka te'ane gosi, <<Gaba'aanota>> jedhantu, lafa Kana'aan irraa fago’oo irraa akka dhufanee  te'anee sobanee Iyaasu’utti himane. Akka inni isaan woliin wolii galtee nageyaa godhuuf Iyaasuu gaafatane. Gaba'aanoti worra eessaa akka te'ane Iyaasu'uu fi Israa'elooti Waaqa hin gaafanne. Maarre Iyaasuun isaan woliin wolii galtee nageyaa godhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-07.jpg?direct&}}

//Israa'elooti akka Gaba'ooti isaan sobane yennaa beekane aarane. Garuu Waaqa duratti waadaan galane waan jiruuf, wolii galtee isaan woliin godhatane san eegatane. Yennaa dhibii turanee, moototi gosa kolbaa dhibi'ii ta Kana'aan keessa jirtu, Amooroti worri Gaba'aanotaa Israa'eloota woliin wolii galtee akka godhatane dhage'ane. Tanaaf loltoota ifii wolitti qabatanee Gaba'aanota lolane. Worri Gaba’aanotaa qarqaarsa gaafatiisaaf, gara Iyaasu’uu ergatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-08.jpg?direct&}}

//Maarre Iyaasuun loltoota Israa'elootaa wolitti qabee Gaba'oon geyiisaaf halkaan guutuu deeme. Ganama loltoota Amoorotaa adoo hin beekin lolane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-09.jpg?direct&}}

//Guyyaa san Waaqi Israa'elootaaf lole. Amoorota joonjessee dhakaa roobaa gugurdaa buusee Amoorota keessaa nama bacaa ijjeese. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-10.jpg?direct&}}

//Ammalle Waaqi akka Israa'elooti worra Amoorotaa ijjuumaa injifatiisaaf yennaa geettu qabaatanu aduun ol-gbbaa irratti addee takkaa akka turtu godhe. Guyyaa san Waaqi Israa'elootaaf injifannoo guddoo kenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-11.jpg?direct&}}

//Waaqii loltoota san injjifateen duuba, gosi kolbaa Kana’aanotaa Israa’el loliisaaf wolitti qabamane. Iyaasu’uu fi Israa’elooti dhibiin worra san lolanee injifatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-12.jpg?direct&}}

//Lola kanaan duuba kutaa lafa waada’aa san irraa Waaqi gosa tokko tokkoofuu qooda isaan gewu kenne. Achiin duuba Waaqi meessaa isi’ii maratti nageya Israa’eliif kenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-15-13.jpg?direct&}}

//Iyaasuun yennaa dulloome, kolbaa Israa’elii mara wolitti waame. Achiin duuba Iyaasuun Gondooroo Waaqi gaara Siina’aa irratti Israa’eloota woliin godhate saniif kolbaan akka ajajantu qaabachiise. Kolbaan Waaqaaf addatantee le’iisaaf, seerata isaalle hordofatiisaaf waadaa galte. //

**//Dabarsii Kitaaba Woyyichaa:- Iyaasuu 1-24 // **
