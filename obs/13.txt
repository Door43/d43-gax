====== 13. **//Gondooroo Waaqi Israa'eloota woliin godhate// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-01.jpg?direct&}}

//Waaqi Israa'eloota karaa abbaayaa diintu'uutiin sooresseen duuba, goomolee keessa gara gaara, <<Siinaa>> jedhamuu geesse. Kunille gaara Muuseen tusee gubattu irratti dhagge. Kolbaan gaara jalatti dunkaanii ifii dhaabbate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-02.jpg?direct&}}

//Waaqi, <<Yoo anaaf ajajantane, gondooroo tiyyalle yoo eegatane, isin lafa ulfinnaa, hayyoota mooti'ii, kolbaa woyyitiille anaaf hin teetan>> jedhee, Muuse'ee fi kolbaa Israa'eliititti dubbate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-03.jpg?direct&}}

//Kolbaan karaa ayyaantumma’aatiin ifi qopheeffatteen guyyaa sadiin duuba, Waaqi mandiisu’uun, balaqqeetti’iin, aaraan, akkasuma qoonqoo tuultulla’aa guddo’oon gaara Sina’aa irratti gad bu’e. Gaara Siina’aa gubbaa akka bawu ka eeyyamameef Moosee qofa. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-04.jpg?direct&}}

//Achiin duuba Waaqi Gondooroo godhee, <<Ani garbummaa biyya Gibxi’ii jalaa ka isin baase, Mootii Waan Maraa Waaqa keessan; waaqota dhibii hin waaqonfatina’a>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-05.jpg?direct&}}

//Muuseelle’een, <<Ati waaqa dharaa hin hujin; hin waaqonfatinille’e. Sababille ani Mootii Waan Maraa Waaqa hinaaticha. Maqaa kiyya akkasumaan hin waamin. Guyyaa sanbataa woyyoonsiisaaf qaabadhu; guyyaa jaa hujii teeti mara hujadhu. Sababille guyyaan torbeesso’oo guyyaa fooraa ka ati ana ittiin qaabatu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-06.jpg?direct&}}

//Abba’aa fi haadha teetiif ulfinna kenni; hin ijjeesin; hin gaarayyeeffatin; hin hatin; dhara hin dubbatin. Niitii namaa, mana isaa, yookiin waan isaa mara fudhatiisaaf hin gaabbin>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-07.jpg?direct&}}

//Achiin duuba Waaqi ajaja kudhan kana dhakaa baxxicha lama irratti barreessee Muuse'etti kenne. Waaqi seeraa fi seerata bacaa dhibiille isaanii kenne. Kolbaan seerata kana yoo muummessite Waaqi akka isaan eebbisuu fi akka isaan eegu addii kenne. Seerata yoo hin muummessin ammoo Waaqi isaan hin adaba. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-08.jpg?direct&}}

//Waaqi marroo dunkaanii akka isaan hujanuuf barbaaduulle issa kenne. Inni <<Dunkaanii Woldeenna'aa>> jedhee isii waame. Isiin kolloo lama ta gollaa gugurda'aan qoqodante qaddiiyyu. Waaqi achi keessa waan le'uuf, kolloo golla’aan duubaa akka seenuuf ka eeyyamameef hayyicha guddaa qofaayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-09.jpg?direct&}}

//Nami seera Waaqaa hin muummessine marti addee ciinc'aa ta dunkaanii woldeenna'aa dura jirtutti horii Waaqaaf ciincaa dhikeeffamu fiduu hin dande'aayyu. Hayyichi horii san qalee addee ciinca'aa irratti guba. Dhiigi horii ciinca'aaf dhikeeffamee sun cubbuu namichaa gollee namicha san Waaqa duratti hin qulqulleessaayyu. Waaqi akka isaan hayyoota isaaf te'anu, obboleessa Muuse'ee Aaronii fi sanyii Aaronii file. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-10.jpg?direct&}}

//Kolbaan marti ajaja Waaqi isaanii kenne muummesiisaaf, Waaqa qofa waaqonfatiisaaf, akkasuma kolbaa  addaa te'iisaaf wolii galane. Garuu Waaqaa ajajamiisaaf waadaa galanee, yennaa diqqo'oon duuba cubbuu hantuu hujane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-11.jpg?direct&}}

//Muuseen yennaa hedduu qaccee gaara Siina’aa irratti Waaqa woliin dubbataayyu. Kolbaan haga inni deebi’u ossaan eegatiisa hifatte. Tanaaf worqii gara Aaronii fidanee waaqa dharaa akka isaaniif huju gaafatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-12.jpg?direct&}}

//Aaroon fakkii  jabbi'iitiin worqii irraa waaqa dharaa huje. Kolbaan haala towanna’aan ala t'een waaqa dharaa waaqonfatisa jalqadde. Ciincaalle isaaf dhikeessite. Marroo cubbuu isaaniitiif Waaqi guddoo isaanitti aaree, isaan balleessiisaaf yaadate. Garuu Muuseen isaaniif daadimate. Waaqille daadimata isaa dhage'ee isaan hin baalleessine. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-13.jpg?direct&}}

//Muuseen gaara irraa  gad bu'ee waaqa dharaa san yennaa dhagge, guddoo aaree dhakaa baxxicha lama ka ajaji kudhan irratti barreeffame san caccasse. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-14.jpg?direct&}}

//Achiin duuba Muuseen waaqa dharaa san daake; bisaan gubbaa bittinneesse. Kolbaalle obaase. Waaqi kolbaa irratti balaa buusee nami bacaan du'e. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-13-15.jpg?direct&}}

//Muuseen ammalle gaara gubbaa bayee akka Waaqi kolba'aaf araaramuuf daadimate. Waaqai Muuse’eef dhage'ee kolba'aaf araarame. Muuseen dhakaa caccasse san bakka buusiisaaf dhakaa baxxicha dhibii irratti ajaja kudhan san barreesse. // //Achiin duuba Waaqi Israa'eloota gaara Siina'aa irraa gara lafa waada’aa  san sooresse. //

**//Dabarsii Kitaaba woyyicaa: Seera Bayiisaa 19-34.// **
