====== 17. **//Gondooroo Waaqi Daawiti woliin godhate// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-01.jpg?direct&}}

//Saa’ol Israa’elii ka qaraa ti. Inni akkuma kolbaan barbaadde nama dheera’aa fi midhaaga’a. Saa’ol Israa’el bulchiisa jalqabee haga woggaa diqqo’oo mootii dansaa te’e. Achiin duuba garuu nama hamaa Waaqaaf hin ajajanne te’e. Tanaaf Waaqi gaafa tokko nama addaa ka addee Saa’oliititti mootii te’u file. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-02.jpg?direct&}}

//Waaqi Saa’oliin duuba mootii akka te’u dargaggeessa Israa’elicha, <<Daawiti>> jedhamu file. Daawiti tissee hoole’ee nama qachaa Beetelemii ti. Daawiti yennaa adda addaa hoolee abbaa ifii yennaa tissu, neenqaa fi doobbaa hoolee san nyaatiisaaf dhufanu ijjeeseeraayyu. Daawiti nama Waaqatti dhugeeffatu, ka isaafille ajajamu, nam gadhi-laafeesa balcha’a. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-03.jpg?direct&}}

//Daawiti lolticha jaba’aa fi sooressa jabaa te’e. Daawiti adoo dargaggeessaatuu, nama akka malee guddatu, ka  <<Golyaad>> jedhamu woliin wol lole. Golyaad lolticha leenjifame hunnaamessa, ka hojjaan isaalle gara meetirii sadii ti. Waaqi garuu  akka Daawiti Golyaad ijjeesee Israa’eloota baasuuf Daawiti qarqaare. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-04.jpg?direct&}}

//Saa’ol jaalaltii kolbaan Daawitiif qaddu gaabbe. Inni Daawiti ijjeesiisaaf yennaa hedduu wodhaane. Tanaaf, Daawiti Saa’ol duraa baqate. Gaafa tokko Saa’ol Daawiti ijjeesiisaaf barbaade. Inni goda Daawiti isa duraa itti dhokateeru dhaqe. Te’uu malee inni Daawiti hin dhaggine. Daawiti gara Saa’olii waan dhikaateeruuf  silaa isa ijjeesuu hin dande’aayyu; inni garuu isa hin ijjeenne. Daawiti mootii te’iisaaf jedhee akka isa hin ijjeesin, Saa’ol akka mirkaneeffatuuf, addee isa ijjeessu handaara woyaa isaa kute. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-05.jpg?direct&}}

//Dhuma irratti Saa’ol lola keessatti du’e. Daawitille mootii Israa’elii te’e. Daawiti mootii dansa’a; kolbaalle isa jaalatte. Waaqi Daawiti eebbisee, isaaf milkeesse. Daawiti lola hedduu lole; Waaqi akka inni diinota Israa’elii injifatuuf isa qarqaare. Daawiti injifatee Yerusaalem qabatee qachaa ifii guddicha  godhate. Bara Daawiti mootii te’e Israa’el hunnaamettii, dureettii teete. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-06.jpg?direct&}}

//Daawiti Galma Waaqaa ka Israa’elooti marti Waaqa itti waaqonfattuu fi ciincaa isaaf itti dhikeesitu ijaariisaaf barbaade. Kolbaaan gara woggaa 400 dunkaanii woldeenna’aa ta Muuseen hujetti Waaqa Waaqonfataa ciincaa isaaf dhikeessitiiyyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-07.jpg?direct&}}

//Waaqi ammo ergaa tana itti himee raagicha Naatan gara Daawitii ergee, <<Ati nama woraanaa waan teeteef, Galma kana anaaf hin ijaartu. Ilma keetitti anaaf ijaara. Garuu ani guddisee si hin eebbisa. Sanyii teeti keessaa tokko kolbaa tiyya irratti Mootii te’ee haga bara baraa hin bulcha>> jedhe.  Ka haga bara baraa bulchu sanyii Daawitii keessaa Masiihii qofa. Masiihiin ka Waaqaan filatame.  Gosa biyya adda addaa ka cubbuu isaanii jalaa baasu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-08.jpg?direct&}}

//Daawiti yennaa jechoota kana dhage’e, Waaqi ulfinna  guddaa kanaa  fi eebba bacaa kana isaa kenniisaaf waadaa waan galeef, yoosuma Waaqa galateefatee leellifate.  Daawiti Waaqi waan kana yoomi akka tolchu hin beenne. Garuu dhugaan jirtu, Israa’elooti Masiihiin adoo hin dhufiniin dura bara hedduu gara woggoota 1000 eegatiisa malaneeyyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-09.jpg?direct&}}

//Daawiti haqaa fi addatamumma’aan woggaa hedduu bulche. Waaqille isa eebbise. Te’uu malee dhuma jiruu ifiititti cubbuu hantuu Waaqa irratti huje. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-10.jpg?direct&}}

//Gaafa tokko loltooti Daawitii marti mana ifiitii bayanee deemanee yennaa lolanu, Daawiti mana mootumma’aa ka ifiitiin ala jiruun namittii miidhadduu nafa ifii dhiqattu takka dhagge. Maqaan isi’ii Baat-Sheeba'a. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-11.jpg?direct&}}

//Daawiti addee fuula ifii isii irraa deebifatu, gara isaa akka dhuttuuf nama itti erge. Inni isii woliin rafee gara manaa isii deebise. Guyyaa diqqa’aan duuba Baat-Sheebaan, <<Ani gadhaachoweera>> jettee Daawititti nama ergite. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-12.jpg?direct&}}

//Dhirsi isiˈii Uriiyaan loltoota Daawitii worra lolaan beekkamane keessa nama tokko. Daawiti akka Uriiyaan lola irraa deebi’ee niitii ifii woliin jiraatu  isa waanisise. Uriiyaan ammo <<Loltooti lola irra jiranuun ani gara manaa hin deeb’u>> jedhe. Daawiti deebisee Uriiyaa gara lolaa ergee akka inni duˈuuf addee lolli jabaan jirutti akka inni duratti isa dabarsu ajajaa loltootaatitti hime. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-13.jpg?direct&}}

//Uriiyaan ijjeefameen duuba Daawti Baat-Sheebaa fuudhe. Duuba irra isiin ilma Daawitiif deette. Waaqi waan Daawiti huje kanaaf guddoo aare. Cubbuun Daawiti huje hantuu akka teete akka isatti himuuf, Waaqi raagicha Naatan erge. Daawiti cubbuu ifii faaciˈatee, Waaqi isaaf araarame. Daawiti bara jiruu ifii ta isaaf hatte keessatti yennaa rakkinna hama’aa keessalle’etti Waaqa hordofee isaaf ajajame. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-17-14.jpg?direct&}}

//Te’uu malee, cubbuu Daawiti hujeef adabbii itti teetee ilmi isaa duˈe. Bara jiruu Daawitii ta hatte keessalle’etti maatii isaa irratti lolli jiraayyu. Hunni Daawitii guddoo dadhadde. Daawiti Waaqaaf addatamaa teˈuu dhabulle, Waaqi ammalle waadaa galeef addatamaˈa. Duuba irra Daawitii Baat-Shebaa irraa ilma uumate; maqaa isaalle Solomoon baasane. //

**//Dabarsii Kitaaba Woyyichaa:- 1Samuel 10:15-19; 24; 31; 2Samuel 5:7; 11-12. // **
