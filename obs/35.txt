====== 35. **//Dabarsii Abbaa Gdhi-laafessaa// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-01.jpg?direct&}}

//Gaafa tokko Yesuus worra qaraxa guuruu fi worra isa dhage’iisaaf  wolitti dacha’e cubbaaleyyii dhibii barsiise**.** //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-02.jpg?direct&}}

//Sooressitooti dhugeeffanna’aa tokko tokko ka achi jiranulle yennaa Yesuus worra cubbaaleyyii jaalota godhatee fudhatu dhaggane; isaan woli wolii isaaniitiin dudubbataa isa tuffatiisa jalqabane. Tanaaf, Yesuus dabarsii tana isaanitti hime: //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-03.jpg?direct&}}

//<<Namichi ijoolle lamaan qabu tokko jiraayyu. Mucaan isaa diqqaan, abbaa ifiitiin, <Abbaa kiyya, karra teeti keessaa qooda kiyya naa kenni!> jedhe. Tanaaf, abbaan isaa karra ifii ijoollee ifii lamaaniif qoode. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-04.jpg?direct&}}

//<<Mucaan diqqaan yoosuma waan qabu mara guuratee, biyya fagoo dhaqee, cubbuu keessa le’aa karra ifii balleesse.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-05.jpg?direct&}}

//<<Achiin duuba biyya mucaan diqqaan jiru sanitti beelti jadduun galte; inni beesee sagalee ittiin bitatu hin qabuuyyu. Tanaaf, inni hujii ifii ta booyyee nyaachisiisaa callaa qabate. Akka malee rakkatee, waan beeloteef, sagalee booyyeen nyaattu  nyaatiisaaf halcheeyyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-06.jpg?direct&}}

//<<Dhuma irratti, mucaan diqqaan sun, <Isii ani maan hujiisatti jira? Tajaajiltooti abbaa kiyyaa marti waan nyaatanu bacaa qaban; ani ammoo beelodheera; gara abbaa kiyyaa deebi’ee deemee, tajaajiltoota isaa keessaa tokko akka na godhatu isa kadhadha> jedhee yaadate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-07.jpg?direct&}}

//Tanaaf, mucaan diqqaan sun gara mana abbaa ifii deebi’iisa jalqabe. Adoo inni fagoo jiruu abbaan isaa isa dhaggee isaaf gadde. Gara ilma ifii rukkisee, hanqafatee sunqate**.** //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-08.jpg?direct&}}

//<<Mucaan isaalle, <Abbaa kiyya, ani Waaqaa fi si abbaa kiyya yakkeera. Ani ilma keeti jedhamiisaaf hin malu> jedhe.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-09.jpg?direct&}}

//<<Abbaan isaa ammoo tajaajiltoota ifii keessaa tokkochaan**, <**Ariifadhuu dhaqiitii woyaa, woyaa mara caaltu fidiitii, mucaa kiyyatti uffis! Quba isaatitti qubee, miila isaatitti kophee keyi. Achiin duuba akka nyaannee gammannu, dardara cooma fida’aatii qala’a; sababille mucaan kiyya kun du’eeraayyu; amma jiraataa te’eera. Badeeraayyu; amma ammoo dhaggameera. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-10.jpg?direct&}}

//<<Tanaaf, namooti gammadiisa jalqabane. Adoo hin turin, obboleessi isaa angafichi addee fichaa keessa hujiisatti jiruu gara manaa dhufee, weeddu’uu fi sirba dhage’ee, <Maanti te’iisatti jira?> jadhee baasefate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-11.jpg?direct&}}

//<<Obboleessi isaa angafichi obboleessi isaa waan dhufeef, namooti akka gammadiisatti jiranu yennaa beeke, waan guddoo mufateef, gara manaa seeniisa didate. Abbaan isaa gad bayee akka inni isa woliin gammadu isa kadhate. Inni ammoo didate.//

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-12.jpg?direct&}}

//<<Mucaan angafichi abbaa ifiitiin, <Ani woggoota tana mara addatamumma’aan si’iif hujeera! Ani ijjumaa ajaja keeti hin didane; akka ani hari’aa tiyya woliin gammadu ilmee re’e’ee takkattiille anaaf hin kennine. Ilmi keeti ka cubbuu hujiisaan karra teeti balleesse kun ammoo, gara manaa yennaa inni dhufe, dardara cooma isaaf qalte!> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-35-13.jpg?direct&}}

  - //<<Abbaan isaa deebisee, <Ilma kiyya, ati yennaa mara ana woliin jirta; wonni ani qabu marti teeti. Gammad keenna ammoo sirri’i; sababille obboleessi keeti du’eeraayyu; amma ammoo jiraataa te’eera; badeeraayyu; amma ammoo dhaggemeera!> jedhe>> jedhe.//
**//Dabarsii Kitaaba Woyyichaa:- Luqaas 15:11-32.// **
