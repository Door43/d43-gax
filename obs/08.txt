====== 8. **//Waaqi Yoseefii fi Maatii Isaa Hambure // **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-01.jpg?direct&}}

//Abbaan isaanii Yaaqoob obboleeyyan mara keessaa irra caalaa Yoseef waan jaalatuu fi mootii isaanii akka te'u Yoseef oojjuu waan oojjateeruuf, obboleeyyan isa jibbite. Yoseef yennaa gara obboleeyyan ifii dhufe isa fuutee daldaltoota garbolee daldaltutti isa gurgurte. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-02.jpg?direct&}}

//Woggaa heddu'uun duuba Yaaqoob yennaa dulloome, obboleeyyan Yoseefii ta horii tissiisatti jirtu akka inni dhaqee isaan ilaalu ilma jaalatu Yoseef erge. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-03.jpg?direct&}}

//Obboleeyyan Yoseefii gara manaa adoo hin deebi'iniin dura woyaa Yoseefii tarsaasanee dhiiga re'e'eetitti cuuphane. Achiin duuba woyaa Yoseefii ta dhiigatti cuuphante san Abba'atti mudhisane. Innille bineensi Yoseef akka ijjeese herrege. Inni guddoo gadde. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-04.jpg?direct&}}

//Daldaltooti garboole'ee gara biyya Gibxi'ii Yoseef geessite. Yennaa san biyyi Gibxi'ii biyya guddittii hunnaamettii laga Nayilii biratti dhaggantu. Daldaltooti garboole'ee Yoseef garba gootee qondaala mootichaa dureessatti gurgurte. Yoseefille goottaa ifii miidhassee tajaajile. Waaqille Yoseef eebbise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-05.jpg?direct&}}

//Niitiin goottaa isaa Yoseef woliin rafiisa barbaadde. Yoseef ammoo, <<Waaqa duratti cubbuu hin huju>> jedhee isii woliin rafiisa didate. Tanaaf isiin aartee dharaan isa waan himatteef inni qabamee mana hidha'aatitti hidhame. Inni mana hidha'aa keessalle'etti Waaqaaf addatamaa te'ee waan dhaggameef, Waaqi isa eebbise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-06.jpg?direct&}}

//Yoseef nama qulqulluu te'ulle woggaa lamaan duuballe manuma hidha'aa keessa jiraayyu. Halkaan tokko mootichi Gibxi'ii Fara'oon oojjuu lama ta akka malee isa rakkittu takka oojjate. Gorsitoota isaa keessa nami tokkolle hiikkaa oojjuu isaa isatti himiisa hin dandeenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-07.jpg?direct&}}

//Waaqi dandeettii oojjuu hiikiisaa Yoseefiif kenneera. Taanaaf, Fara'oon mana hidha'aa keessaa, Yoseef gara isaa akka fidanu godhe. Yoseef oojjuu isaa isaaf hiikee, <<Waaqi bara misaa torba hin fida; itti aanee barri torba bara beelaa hin te'a>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-08.jpg?direct&}}

//Fara’oon guddoo Yoseef baaseffatee, biyya Gibxi’ii mara keessatti ifitti aanisee bulchaa lammeesso’oo godhee isa muude. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-09.jpg?direct&}}

//Yoseefille kolbaan bara misaa torba san keessa midhaan bacaa gootara’atti akka kuufattu hime. Barri beelaa torba yennaa dhufe, Yoseef gootaraa keessaa midhaan kolba’atti waan gurgureef, kolbaan waan nyaatanu bacaa dhaggatte. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-10.jpg?direct&}}

//Hameenni beelaa biyya Gibxi’ii keessa qofa adoo hin te’in lafa kaana’aanii ka Yaaqoobii fi maatiin isaa keessa le’anu keessalle hin jiraayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-11.jpg?direct&}}

//Yaaqob ijoolleen isaa midhaan akka bittuuf gara biyya Gibxi’ii erge. Obboleyyan Yoseefii midhaan bitiisaaf dhaqanee isa dura yennaa dhaabbatane akka inni Yoseef te’e hin beenneeyyu. Inni ammoo isaan beekeera. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-12.jpg?direct&}}

//Yoseef obboleeyyan ifii haga ilaaleen duuba akka isaan jijjiiramaneeranu ilaaliisaaf, <<Hin sodaatina’a! Ani obboleessa keessan Yoseef. Isin akka ani garbicha te’u yennaa na gurgurtanu waan hanutuu na godhiisaaf wodhaantane; garuu Waaqi addee waan hantuu sanii waan dansaa godheera; kowa’aatii biyya Gibxi’ii keessa le’a’a; anille isinii fi maatii teessaniif waan barbaachisu hin dhikeessa>> jedheen. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-13.jpg?direct&}}

//Obboleeyyan Yoseefii gara biyya ifii deebi’anee, Yoseef lubbu’uun akka jiru abbaa ifii Yaaqoobitti yennaa himane, Yaaqoob guddoo gammade. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-14.jpg?direct&}}

//Yaaqoob jaarsa te’ulle maatii ifii mara woliin gara biyya Gibxi’ii dhaqee achi le’e. Yaaqoob adoo hin du’iniin dura ilmaan ifii tokko tokkoo isaaniituu eebbise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-08-15.jpg?direct&}}

//Waadaan Waaqi Abrahaamiif gale gara Yisihaaqii dabare. Achiin duuba gara Yaaqoobii, itti aanee gara ilmaan Yaaqoobii kudhanii lamaatii fi maatii isaanii dabare. Sanyiin imaan kudhanii lamaa gosa Israa’elii kudhanii lama. //

**//Dabarsii Kitaaba woyyichaa :- Seera Dadaa 37-50 // **
