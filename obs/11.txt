====== 11. **//Irra Dabarama// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-01.jpg?direct&}}

//Fara'oon Israa'eloota yoo gad hin lakkisin ilmaan isaanii angafootaa, korommii isaanii angafootaa fi korbeeyyii isaanii angafoota mara akka ijjeesu Waaqi isa ifi eegachiise. Fara'oon tana yennaa dhage'e, ammalle dhugeeffatee Waaqaaf ajajamiisa didadate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-02.jpg?direct&}}

//Waaqi worra isatti dhugeefate angafoota dhiira isaanii mara hamburiisaaf karaa qopheesse. Maatiin tokko tokkooyyuu buruusoo hoole'ee ka fafa hin qanne filattee qalatte. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-03.jpg?direct&}}

//Waaqi Isaraa'elooti dhiiga buruusoo hoole'ee san mikikkila balbala ifii akka dibatanu, foon isaaniillee woddanee daabboo uukoo hin qanne woliin ariiti'iin akka nyaatanu isaanitti hime. Yennaa nyaatane biyya Gibxi’ii keessaa bayiisaafille akka qophonu isaanitti hime. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-04.jpg?direct&}}

//Israa'elooti waan Waaqi akka isaan hujanu ajaje mara hujane. Halkan wodhakkaa Waaqi biyya Gibxi’ii mara keessa dabaree angafoota dhiiraa mara ijjeese. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-05.jpg?direct&}}

//Mikilli mana Israa'elootaa marti dhiigi itti dibameeraayyu. Maarre Waaqi mana isaanii irra dabare. Nami mana keessa jiru marti hafe. Israa'elooti sababa dhiiga buruusoo korbeessa hoole'ee saniif hafane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-06.jpg?direct&}}

//Gibxooti ammoo Waaqatti hin dhugeeffanne. Ajaja isaalle hin muummessine. Tanaaf, Waaqi mana isaanii irra hin dabarre. Waaqi angafoota dhiira  Gibxitootaa mara ijjeese. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-07.jpg?direct&}}

//Angafoota dhiiraa ka mana hidha'aa jiranuu kaasee haga angafa dhiiraa ka Fara'ooniititti angafooti  dhiiraa ka worra Gibxi'ii marti du'ane. Gguddoo waan gaddaneef, worri biyya Gibxi’ii keessa jiru hedduun  iyyanee booyane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-11-08.jpg?direct&}}

//Halkanuma san Fara'oon Muuse'ee fi Aaron waansisee, <<Ammuma Israa'eloota fudha'aatii deema>> jedheen. //

**//Dabrsii Kitaaba Woyichaa:- Seera bayiisaa:-11:1-2;32. // **
