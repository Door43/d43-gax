====== 16. **//Worra Bilisa Baasu// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-01.jpg?direct&}}

//Iyaasuun du’een duuba Israa’elooti Waaqaaf hin ajajanne. Worra Kana’aanota ka hafanelle odduu ifiitii hin baanne yookiin seera Waaqaatiifille hin ajajanne. Israa’elooti Waaqa dhugicha Mootii Waan Maraatiif ajajamiisa irra waaqota Kana’aanotaa Waaqonfatiisa jalqabane. Israa’elooti Mootii hin qabanuuyyu. Tanaaf tokko tokkoon isaanii waan isaanitti sirrii te’ee mudhate mara godhaniiyyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-02.jpg?direct&}}

//Israa’elooti Waaqaaf ajajama dhabiisa ittuma waan deemaneef, Waaqille akka diinni isaanii isaan miidhu godhe. Diinoti isaanii kun meya Israa’elootaa mara hatane; karra isaanii mara balleesanee, isaan keessaa nama bacaa ijjeesane. Israa’elooti Waaqaaf ajajamiisa dhabiisaa fi diinotaan miidhamiisaan bara hedduu dabarsaneen duuba araara galane; Waaqille diina isaanii jalaa akka isaan baasu gaafatane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-03.jpg?direct&}}

//Achiin duuba Waaqi worra diina isaanii jalaa bilisa isaan baasu isaaniif erge; lafalle’eef nageya kenne. Te’uu malee namooti ammalle Waaqa dedhanee, waaqa dharaa waaqonfatiisa jalqabane. Maarre Waaqi diinota isaanii Miidiyaanota ka dhikoo isaanii jiranu isaanitti kaasee, akka isaan injifatamanu godhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-04.jpg?direct&}}

//Worri Miidiyaanotaa woggaa torba midhaan Israa’elootaa mara fudhatane. Israa’elooti guddoo sodaatane. Isaan dhooqaa fi goda keessatti waan dhokataneef worri Miidiyaanotaa isaan dhaggiisaaf hin dandeenne. Dhuma irratti akka isaan qarqaaruuf gara Waaqaa iyyane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-05.jpg?direct&}}

//Gaafa tokko Israa’elichi, <<Gede’oon>> jedhamu tokko worri Miidiyanootaa akka midhaan isa irraa hin fudhaneef, dhokatee midhaan dhowatiisatti jiraayyu. Ergamaan Waaqaa gara isaa dhufee, <<Ati lolticha jaba’a, Waaqi si woliin jira. Dhaqiitii Isaraa’eloota worra Miidiyaanotaa jalaa baasi>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-06.jpg?direct&}}

//Abbaan Gede’oonii addee ciinca’aa ta waaqa dharaa  waaqonfatiisaaf qophootte qabaayyu. Waaqi akka Gede’oon addee ciinca’aa san diiguuf dubbate. Te’uu malee Gede’oon kolbaa waan sodateef , haga latti dhiitu eege. Achiin duuba addee ciinca’aa san diigee caccassee butuche. Addee san biratti addee ciinca’aa haareya Waaqaaf hujee irratti ciincaa Waaqaaf  dhikeesse. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-07.jpg?direct&}}

//Boruyyaa ganama kolbaan akka nami tokko addee ciinca’aa ta isaanii diige dhaggitee aarte. Gede’oon ijjeesiisaaf gara mana isaa dhaxxe. Abbaan Gede’oonii ammoo kolba’aan, <<Isin maaf Waaqa keessan qarqaariisaaf wodhaantan? Inni Waaqa yoo te’e, innummaan ifi irraa dhooggatuu ti>> jedhe. Inni waan akkas jedheef kolbaan Gede’oon hin ijjeenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-08.jpg?direct&}}

//Worri Miidiyaanotaa ammalle Israa’eloota saamiisaaf dhufane. Isaan laakkosaan bacaa waan te’aneef eennulle isaan laakkowee hin dandeenne. Gede’oon worra Miidiyaanotaa loliisaaf Israa’eloota waame. Waaqi karaa isaatiin akka Israa’eloota qarqaaru mirkaneeffatiisaaf, Gede’oon beessisa lama Waaqa gaafate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-09.jpg?direct&}}

//Beessisa qaraatiif, hoccuu laffatti afee fixeensa ganamaa hoccuu qofa irra malee lafa irratti akka hin dhagganne gaafate. Waaqille akkasuma godhe. Halkaan itti aanu Gede’oon, <<Latti fixeensa ganamaatiin jiitee hoccuun ammoo godduu akka teetu>> jedhee Waaqa gaafate. Waaqille akkasuma godhe. Beessis kun lamaan Waaqi karaa isaatiin Israa’eloota worra Miidiyaanotaa jalaa akka baasu Gede’ooniif ragaa te’ane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-10.jpg?direct&}}

//Loltooti Israa’elootaa 32,000 gara Gede’oonii dhufane. Waaqi ammoo <<Laakkossi isaanii bacateera >> jedheen. Tanaaf, Gede’oon loltoota loliisa sodaatte 22,000 gara mana isaaniititti deebise. Waaqi ammalle Gede’ooniin, <<Laakkossi loltootaa bacateera>> jehdeen. Tanaaf, Gede’oon loltoota 300 hamburee kaan gara mana isaanii deebise. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-11.jpg?direct&}}

//Galgala san Waaqi Gede’ooniin, <<Gara quttuma loltoota worra Miidiyaanotaa dhaqi;  waan siin jedhanulle  yoo dhageette isaan hin sodaatin>> jedhe. Tanaaf Gede’oon galgaluma san gara quttuma loltoota Miidiyaanotaa dhaqe. Loltichi Miidiyaanotaa tokko, loltooti Gede’oonii loltoota Miidiyaanotaa akka injifatane lansa ifiititti himuun oojju’uun dhagge. Gede’oon yennaa tana dhage’e Waaqaaf sagade. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-12.jpg?direct&}}

//Achiin duuba Gede’oon gara loltootaa deebi’ee tokko tokkoo isaaniitiif tultullaa gaafaa, okkote’ee fi xassaa kenneefi. Isaan adoo loltooti Miidiyaanotaa rafaneeranuu  addee quttuma isaanii marsane. Loltooti Gede’oonii 300 okkotee san keessaa xassaa qabaniiyyu; loltooti Miidiyaanotaa garuu ifa xassa’aa dhaggiisa hin dandeenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-13.jpg?direct&}}

//Achiin duuba loltooti Gede’oonii marti yennuma takkattii okkotee ifi cassanee yennuma san xassaan dedhuma ife. Isaan tuultullaa ifii ka gaafaa afuufaa, <<Loltoota Waaqaatii fi loltoota Gede’oonii>> jedhaa iyyane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-14.jpg?direct&}}

//Waaqi Miidiyaanota joonjessee, isaan woluu ijjeesane. Yennuma san Israa’elooti mana ifii jiranu waamamanee Miidiyaanota ari’iisaaf wol qarqaarane. Isaan worra bacaa ijjeesanee worra hafelle lafa Israa’elootaa keessaa ari’ane. Yennaa san loltooti Miidiyaanotaa 120,000 du’ane. Waaqille Israa’eloota hambure. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-15.jpg?direct&}}

//Kolbaan akka Gede’oon mootii isaaniif  te’u barbaadde. Gede’oonii ammoo tana akka godhanu isaaniif hin eeyyanne. Garuu tokko tokkoon isaanii worqii Miidiyaanota irraa fudhatane akka isaaf fidanu gaafate. Kolbaalle worqii bacaa Gede’ooniif kennite. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-16.jpg?direct&}}

//Gede’oon  worqii san baqsiisee woyaa addaa ta akka hayyichi guddaan uffatuu godhee huje. Kolbaan ammoo woyaa san akka waaqa dharaa waaqonfatanu waaqonfatiisa jalqadde. Tanaaf Waaqi akka diinni isaan injifatu godhee isaan adabe. Dhuma irratti kolbaan ammalle akka Waaqi isaan baasuuf gaafatte. Waaqille qarqaaraa dhibii isaaniif erge. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-17.jpg?direct&}}

//Haalli akkasii yennaa hedduu itti dedddeebi’ame. Israa’elooti cubbuu hujan; Waaqi isaan adaba; isaan deebi’anee araara galan; Waaqille isaan baasiisaaf nama isaan baasu isaaniif erga. Waaqi woggaa hedduu Israa’eloota baasiisaaf,  worra bilisa isaan baasu hedduu ergeera. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-18.jpg?direct&}}

//Dhuma irratti,  kolbaan akkuma gosi biyya adda addaa ta dhibiin qaddu, Waaqi mootii akka isaaniif kennu gaafatane. Ka yennaa lolaa isaan sooressu, mootii dheeraa jabaa akka qabaatanu barbaadane. Waaqi ammoo gaaffii tana hin jaalanne. Te’uu malee akkuma isaan barbaadane mootii isaaniif kenne. //

**//Dabarsii Kitaaba Woyyichaa:- Kitaaba Moototaa 1-5: 6-8 // **
