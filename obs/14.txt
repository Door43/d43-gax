====== 14. **//Goomolee Keessa Joonjane// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-01.jpg?direct&}}

//Waaqi Gondooroo isaan woliin godhate kutaa teete saniitiif akka ajajamanu Israa'elootatti himeen duuba  gaara Siina'aa irraa deemane. Waaqi  gaara Siina'aa irraa gara lafa,  <<Kaana'aan>> jedhantuu  ta isaanii kenniisaaf waadaa galee kolbaa Israa’elii sooressuu jalqabe. Utubaan duumensaa isaaniin dura gara Kaana'aanii deeme; isaanille isa gula deemane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-02.jpg?direct&}}

//Waaqi lafa waadaa gale san sanyii isaaniitii kenniisaaf, Abrahaamiif, Yisihaaqii fi Yaaqoobiif waadaa galeeraayyu. Garuu lafa san amma gosi adda addaa bacaan keessa le’aniiyyu. Isaan, <<Kaana'aanota>> jedhaman. Kaana'aanoti Waaqa hin waaqonfanne yookiin isaaf hin ajajanne; waaqa dharaa waaqonfatanee waan bacaa godhane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-03.jpg?direct&}}

//Waaqi Israa’elootaan, <<Kaana’aanota  lafa waada’aa keessa jiranu san mara balleessa’a. Isaan woliin nageya hin qabaatina’a; durra isaaniille hin fuudhina'a. Waaqota dharaa ka isaanii ijjumaa balleessa’a. Isin yoo anaaf hin ajajamin, addee ana waaqonfattanu waaqota isaanii waaqinfattan>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-04.jpg?direct&}}

//Israa'elooti meessaa Kaana'aanotaa yennaa geyane, Muuseen gosa Israa'elii tokko tokkooyyuu keessaa namoota kudhanii lama  file. Isaan dhaqanee latti sun attam akka jirtu akka dooyanuuf,  qajeelfama kenne. Namooti kun Kaana'aanoti worra jajjaba yookiin worra dadhabaa akka te'ane dooyuu qabaniiyyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-05.jpg?direct&}}

//Namooti kudhanii lamaa sun guyyaa afurtama Kaana'aan keessa deddeemanee deebi'anee dhufane. Isaan, <<Latti lafa dansa’a! Midhaan isii keessaalle baca'a>> jedhanee kolba'atti himane. Worri dooye kudhan ammoo, <<Qachooti lafa sanii jajjaba’a; namootille hunnaameyyi’i; yoo isaan lolle dhugumaan nu hin injifatan; nu hin ijjeesanille'e>> jedhane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-06.jpg?direct&}}

//Yoosuma worra dooye san keessaa lamaan Kaalebii fi Iyaasuun, <<Namooti Kaana'aanii dhedheeraa jajjaba te'iis isaanii dhuga'a; garuu nuuti dhugumaan isaan hin injifanna! Waaqi nu'uuf hin lola!>> jedhane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-07.jpg?direct&}}

//Namooti ammoo Kaalebii fi Iyaasuu hin dhageenne. Muuse'ee fi Aaronitti aaranee, <<Addee badduu tana maaf nu fiddane? Woraanaan addee dhumanu, niitoleen teennaa fi ijoolleen teenna garboolee addee te'anu biyya Gibxi’ii keessa adoo leenee nuu woyya>> jedhane. Kolbaan nama gara biyya Gibxi'ii isaaan deebisu sooressa addaa filatiisa barbaadde. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-08.jpg?direct&}}

//Waaqi guddoo aaree gara dunkaanii woldeenna’aa dhufee, <<Isin waan ana finqiltaneef, kolbaan marti goomolee keessa joonjiisatti jirti. Iyaasu’uu fi Kaaleb malee namooti  woggaan isaanii diddamaatii fi diddamaan olii marti achumatti du’an malee lafa waada’aa sanitti hin galanu>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-09.jpg?direct&}}

//Kolbaan tana yennaa dhageette, cubbuu waan hujjeef gaddite. Meya woraanaa qabattee worra kana’aanii loliisaaf ideente. Muuseen marroo Waaqi isaan woliin hin jirreef akka isaan hin ideenne ifi eeggachiise. Garuu isaan isa hin dhageenne. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-10.jpg?direct&}}

//Waaqi isaan woliin gara lolaa hin dhandhe. Tanaaf injifatamanee nami bacaan ijjeefame. Achiin duuba Israa'elooti Kana'aan irraa deebi'anee goomolee keessa woggaa afurtama joonjane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-11.jpg?direct&}}

//Woggaa afurtama kolbaan Israa'elii goomolee keessa joonjite san Waaqi sagalee isaan nyaachise. Inni ol-guubbaa irraa Sagalee, <<Mannaa>> jedhantu isaanii kenne. Foon akka nyaatanuufille addee quttuma isaaniititti barbadi-kille isaaniif erge. Yennaa san mara keessa woyaan isaaniitii fi copheen isaanii akka dhumanne godhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-12.jpg?direct&}}

//Waaqi karaa bilbaasaatiin rassaa keessaa bisaan isaaniif baase. Tun marti isaaniif teetulle, kolbaan Israa'elii Waaqaa fi Muusee irratti aaranee gungumane. Te'uu malee Waaqi waadaa Abraamiif, Yisihaaqii fi Yaaqoobii galeef ammalle addatama'aayyu. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-13.jpg?direct&}}

//Yennaa dhibii yennaa kolbaan ijjumaa bisaan hin qanne, Waaqi Muuse'een, <<Rassaa kannatti dubadhu; isa keessaa bisaan hin baya>> jedhe. Te’uu malee Muuseen addee rassa'atti dubbatu, ule'een yennaa lama waan dhoweef, kolbaa mara duratti Waaqa salphise. Nami marti akka unuuf, rassaa keessaa bisaan baye. Waaqi ammoo Muuse'etti aaree, <<Lafa Waadaa sii gale sanitti hin galtu>> jedheen. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-14.jpg?direct&}}

//Israa'elooti woggaa afurtama goomolee keessa joonjaneen duuba, worri Waaqa irratti finqila kaase marti du'e. Achiin duuba Waaqi ammalle kolbaa gara meessaa lafa waadaa isaaniif galee san sooresse. Amma Muuseen dulloomeera; tanaaf kolbaa sooressiisaan akka isa qarqaaruuf, Waaqi Iyaasuu file. Ammalle Waaqi gaafa tokko raagaa dhibii ka akka Muuse'ee ergiisaaf, Muuse'eef waadaa gale. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-14-15.jpg?direct&}}

//Achiin duuba Waaqi akka Muuseen lafa waada’aa san dhagguu dande’u, qaccee gaaraa gubbaa akka ol bawu itti dubbate. Muuseen lafa waada’aa san dhagge; garuu akka inni achi galu Waaqi isaaf hin eeyyanne. Achiin duuba Muuseen du’e; Israa’elooti guyyaa soddoma gaddane. Iyaasuun sooressa haareya isaaniif te////’////e. Iyaasuun nama Waaqatti dhugeeffatee isaaf ajajame waan te////’////eef sooressa dansaa te////’////e. //

**//Dabarsii Kitaaba Woyyichaa:- Seera Bayiisa 16-17; Seera Laakkossaa 10-14; 20; 27; Seera Keessa Deebii 34. // **
