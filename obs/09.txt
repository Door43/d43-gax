====== 9. **//Waaqi Muusee Waame// **  ======

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-01.jpg?direct&}}

//Yoseef du'een duuba worri isaa marti biyya Gibxi'ii keessa turane. Isaanii fi sanyiin isaanii marti bara hedduu achuma le'anee, ijoollee bacaa horane. Isaan, <<Israa’eloota>> jedhamanee waamamane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-02.jpg?direct&}}

//Woggaa dhibbaan duuba laakkossi Israa’elootaa bacataa deeme. Worri biyya Gibxi'ii Yoseefii fi waan inni isaan qarqaariisaaf godhe mara dedhane. Israa'elooti bacataa waan deemaneef, worri biyya Gibxi'ii isaan sodaatane. Tanaaf, Fara'oon ka bara san biyya Gibxi'ii bulchu Israa'eloota worra Gibxi'iitiif garboota tolche. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-03.jpg?direct&}}

//Worri biyya Gibxi'ii akka Israa'elooti ijaarsa dhaka'aa ijaartu, qachaa mara akka ijaartu godhanee hunnaan isaan hujjisiifatane. Hujiin ulfaattuuun jiruu isaanii rakkitte. Waaqi ammoo isaan eebbisee ijoollee bacaa horane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-04.jpg?direct&}}

//Fara'oon akka worri Israa'elootaa ijoollee bacaa qabanu dhagge. Maarre Fara'oon ijoollee Israa'elootaa keessaa ilmaan dhiiraa yennaa dhalattu laga Nayiliititti gatamanee akka ijjeesanuuf kolbaa ajaje. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-05.jpg?direct&}}

//Dubartiin Israa'elittiin takka ilma deette. Isi'ii fi dhiris isi'ii haga dande'ane ilma ifii san dhossane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-06.jpg?direct&}}

//Haadhaa fi abbaan mucaa sanii mucaa san dhossuu yennaa dadhabane, saaxaraa bisaan gubbaa deemu keessa keyanee, akka inni hin ijjeefanne hamburiisaaf, alaaduu laga Nayilii fuula dura bisaan gubbaa gad lakkisane. Intalti Fara'oonii saaxaraa san dhaggitee, waan keessa jiru ilaalte. Obboleettiin isaa angafittiin waan isa godhanu ilaaliisaaf, dhiko'otti ilaalaa turte. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-07.jpg?direct&}}

//Intalti Fara'oonii saaxaraa yennaa dhaggite waan keessa jiru ilaalte. Annuu yennaa dhaggite, akka ilma ifii tolfattee fudhatte. Isiin annuu san nama isi'iif guddisu dubartii Israa'elittii takka qaxaratte. Isiin dubartii qaxaratte san akka isiin haadha annuu sanii teete hin beenne. Annuun sun guddatee harma haadhaa hodhiisa yennaa lakkise, dubrtittiin sun annuu san gara intala Fara'oonii deebitte. Isaan, <<Muusee>> jedhanee maqaa isa baasane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-08.jpg?direct&}}

//Gaafa tokko Muuseen guddateeruun, nami Gibxi'ii tokko nama Israa'elii garbicha tokko dhaanuun dhaggee, gosa ifii Israa'eelicha san akka inni hin duune hamburiisaaf wodhaane. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-09.jpg?direct&}}

//Museen waan nami hin dhagginne itti fakkaattee, nama Gibxi'ii san ijjeesee awwaale. Garuu waan Muuseen godhe san nami tokko dhaggeera. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-10.jpg?direct&}}

//Fara'oon waan Muuseen godhe san yennaa dhage'e, isa ijjeesiisaaf wodhaane.Muuseen ammoo loltoota Fara'oonii jalaa gara lubuu ifii itti hamburatu, biyya Gibxi'iitii gara goomole'ee baqate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-11.jpg?direct&}}

//Muuseen goomolee biyya Gibxi'ii irraa fagoo jirtu keessa tissee hoole'ee te'e. Inni biyya san keessaa niitii fuudhee ilmaan lama uumate. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-12.jpg?direct&}}

//Gaafa tokko Muuseen hoolee adoo tissuu tusee ibiddi keessaa bobe'u takka dhagge. Te'uu malee tuseen sun hin gubanne. Muuseen hubatee ilaaliisaaf gara tusee sanii dhikaate. Inni gara tusee ibiddi keesaa bobe'u sanii yennaa dhikaate, qoonqoon Waaqaa, <<Muuse'e! Addee woyyittii waan dhaabbatteertuuf, miila keeti irraa kophee baafadhu>> jette. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-13.jpg?direct&}}

//Waaqi, <<Ani rakkinna kolbaa tiyyaa waan dhage’eef, gara Fara’oonii si erga. Atille Israa’eloota garbumaa biyya Gibxi’ii jalaa hin baatta. Ani lafa Kana’aanii ta Abrahaamiif, Yisihaaqiif, Yaaqoobiifille kenniisaaf waadaa gale san isaaniif hin kenna>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-14.jpg?direct&}}

//Muuseen, <<Kolbaan eentu si erge?>> jettee yoo na gaafatte, <<Ani maan isaaniin jedha>> jedhee, Waaqa gaafate. Waaqille Muuse'een, <<Ani Waaqa jiruu fi jiraadhu; tanaaf, <Ka jiruu fi jiraatu na ergeera> jedhii isaanitti himi>> jedhe. //

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-09-15.jpg?direct&}}

//Muuseen waan sodaateef gara Fara’oonii dhaqiisa hin barbaanne. Sababille, inni, <<Ani dandeettii dubbatiisaa hin qabu>> jedhee waan herregeef sodaate. Maarre Waaqi obboleessi Muuse’ee Aaron isa woliin deemee akka isa qarqaaruuf isa woliin erge. Fara’oon, <<Ani kolbaa gad hin lakkisu>> akka jedhu, gadhaa isaa akka jabeessu Waaqi Muuse’ee fi Aarootitti hime. //
